i-node
 - 하나의 파일은 하나 가짐
 - 정보 : 종류, 소유권, 액세스모드, 파일 갱신일

데이터 블록
 - 파일에서 데이터 저장 블록
종류
 - 일반		: 
 - 디렉터리	: 트리 구조로 관리
 - 링크		: i-node와 연결
 - 하드링크	: 사본 파일으 만듬
 - 심볼릭 링크	: 바로가기
 - 특수 파일	: 주변장치


리눅스 명령어
 - pwd		: 현재 디랙토리 확인
 - cd		: 파일 위치 이동
 - ls		: 현재 디렉터리 내용 확인
	옵션: -a 모든 종류	-F 파일 종류  -d 디렉터리  -l 심볼릭링크
 - mkdir		: 새 디렉터리 생성
	옵션: -p 중간에도 자동 삭제
 - rmdir 		: 디렉터리 삭제
	옵션: -p 중간에도 자동 삭제
 - rm		: 디렉터리 삭제
 - touch		: 파일 생성
 - cp		: 파일 복사
 - find		: 파일 검색
 - cat		: 파일 내용 출력
